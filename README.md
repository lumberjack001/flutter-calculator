# Flutter Calculator

A simple calculator app built with Flutter that supports basic arithmetic operations.

## Features

- **Addition** (`+`)
- **Subtraction** (`−`)
- **Multiplication** (`×`)
- **Division** (`÷`)
- Decimal point support
- Percentage (`%`)
- Toggle sign (`+/-`)
- All-clear (`AC`)
- Chained operations
- Division-by-zero error handling

## Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) ≥ 3.0.0

### Run the app

```bash
flutter pub get
flutter run
```

### Run tests

```bash
flutter test
```

## Project Structure

```
lib/
  main.dart          # Calculator UI and logic
test/
  widget_test.dart   # Widget tests for all arithmetic operations
```
